# 🏠 부동산 매물 정보 제공 플랫폼 기능 명세

---

## 📌 필수 기능

### 1️⃣ RAG 매물 추천

**설명:** 사용자의 예산, 위치, 평수, 교통, 생활 패턴 등을 기반으로 AI가 유사 매물을 전달.
**세부 기능:**

- 자연어 질의 기반 매물 검색 (“3억 이하 강동구 전세”)
- 벡터 DB 기반 유사도 검색 (pgvector)

---

### 2️⃣ 데이터 수집 및 정제

**설명:** 공공데이터·부동산 API·포털 크롤링을 통해 매물 정보를 자동 수집 및 표준화. (정확한 정보은 미정)
**세부 기능:**

- 주소 및 좌표 정규화
- 중복/허위 매물 필터링
- 거래 유형별 표준화 (매매) -> (전세/월세/분양 등 <b>확장 예정</b>)
- 시세·평균가 산출 및 메타데이터(면적, 평당가, 연식, 방향 등) 정제

---

### 3️⃣ 사용자 관리

**설명:** 이용자 인증, 개인화 서비스, 권한 기반 접근 제어.
**세부 기능:**

- 회원가입/로그인 (oauth2 인증, JWT 토큰 기반 인가)
- 마이페이지: 관심 매물, 최근 본 매물
- 사용자 등급: 일반, 관리자
- 개인정보 수정, 계정 삭제 기능 / (알림 설정<b>확정 예정</b>)

---

### 4️⃣ 지도 기반 매물 확인

**설명:** 지도를 통해 매물의 위치, 시세, 교통 인프라를 직관적으로 탐색.
**세부 기능:**

- 지도 위 매물 핀 표시 (Kakao Map / Naver Map API 中 1)
- 줌 레벨별 필터링 (아파트 단지 / 동 / 지역 단위)
- 편의시설(학교, 병원, 지하철 등) 오버레이 표시
- 매물 클릭 시 상세정보·사진·가격 추이 표시

---

### 5️⃣ 정책 관리

**설명:** 사이트 운영 정책, 이용약관, 공지사항 등을 효율적으로 관리.
**세부 기능:**

- 관리자 페이지에서 정책 등록/수정/버전 관리
- 이용약관·개인정보처리방침 자동 표시
- 공지사항 및 업데이트 내역 게시

---

### 6️⃣ 현황 분석 리포트

**설명:** 관리자 및 사용자에게 시장 현황과 매물 통계를 시각화로 제공.
**세부 기능:**

- 지역별 매물 수, 거래량, 평균가 통계
- 기간별 가격 변동 그래프
- 인기 매물 / 조회수 상위 랭킹 (우리 사이트 기준?? or 데이터 수집해서 처리??)
- 대시보드 형태로 갱신 (데이터 갱신 시점에 맞춰)

---

## 💬 부가 기능

### 1️⃣ 자유게시판

**설명:** 이용자 간 정보 공유 및 커뮤니티 공간 제공.
**세부 기능:**

- 글 작성/수정/삭제, 이미지 첨부
- 좋아요/댓글/신고 기능
- 검열 (음란물, 욕설 등등 차단 정책 수립)

---

### 2️⃣ 매물별 댓글

**설명:** 매물 상세 페이지 내 실거주 후기 및 의견 교류 기능.
**세부 기능:**

- 대댓글, 신고, 정렬(최신순/공감순)
- 검열 (음란물, 욕설 등등 차단 정책 수립)

---

### 3️⃣ 매물별 채팅

**설명:** 사용자 간 실시간 매물별 정보 공유 채널.
**세부 기능:**

- WebSocket 기반 N:M 상담
- 메시지 알림, 대화 로그 저장
- 이미지 전송, 신고/차단 기능
- 검열 (음란물, 욕설 등등 차단 정책 수립)

---

### 4️⃣ 매물 변동 알림 (SMS·메일)

**설명:** 관심 매물의 가격 변동, 거래 완료, 신규 등록 시 실시간 알림 발송.
**세부 기능:**

- SMS, 이메일 3채널 지원
- 알림 조건(가격 변동률, 거래 상태) 커스터마이징
- 수신 동의/해제 관리 (개인정보보호 준수)
